# devbymax — Project Plan

Freelancer portfolio landing page. Sells web development to small businesses (cafes, salons, shops).
Stack: HTML + CSS + Vanilla JS. Dark premium theme. Russian copy. CTA → Telegram.

## Phases

| # | Phase | Files | Done when |
|---|-------|-------|-----------|
| 1 | Scaffold | index.html, css/style.css, js/main.js | CSS vars, fonts, meta/OG tags, blank sections |
| 2 | Header + Hero | index.html, style.css | Sticky blur header, 2-col hero, browser mockup, both CTAs |
| 3 | Social Proof + Problems | index.html, style.css | Marquee ticker, 3 problem cards, gold resolution line |
| 4 | Services | index.html, style.css | 4 service cards with emoji, description, price tag |
| 5 | Cases | index.html, style.css | 2 case cards (problem/solution/metrics), testimonials |
| 6 | Process | index.html, style.css | 5-step zigzag timeline, day badges, numbered gold circles |
| 7 | Pricing | index.html, style.css | 3 tier cards, Популярный badge on Бизнес, feature lists |
| 8 | FAQ + CTA + Footer | index.html, style.css, main.js | Accordion FAQ, glow CTA block, footer |
| 9 | Animations | main.js, style.css | Scroll-reveal (IO), counter animation, all hover states |
| 10 | Mobile | style.css, main.js | 375px audit, hamburger menu, sticky bottom CTA bar |
| 11 | Polish | all | Favicon, PageSpeed > 90, link check, final QA |
